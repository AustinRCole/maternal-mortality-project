Machine Learning Models

Testing and Comparing Multiple Machine Learning Models!

Overview

We split our data into two main subsets for training models, one utilizing data about race, and another using data that did not include race as a feature. We then trained and tested a variety of models, before choosing the top-performing models for each subset.

Linear Regression Model

This model examines the impact of various features on maternal mortality ratio irrespective of race. In doing so, correlations were determined using linear regression analyses and indicated positive and negative relationships.

Based upon the initial heatmap which included all features, a second more focused heatmap was developed. The initial correlation results showed specific features with strong positive and negative correlation as can be seen in heatmap 1. Thus, in heatmap 2 above the analyses indicate that diabetes and premature death had the strongest positive correlations. Other important correlations included positive relationships with physical inactivity, obesity, and low birth weight. Interestingly, medicare coverage also had a moderately strong correlation with MMR. High health status, which is the percentage of women who reported that their health is very good or excellent, had the strongest negative correlation in addition to higher weighted sums of all determinants and health outcomes from the national average. Dental visits also had a moderately strong negative correlation with MMR.

Synopsis Linear Regression Model - All Features

We used Linear regression analysis, which is supervised machine learning, because MMR is a continuous outcome. All features except birth and death were compared against MMR. R-squared for all the features was 0.54, which suggests that together the features only moderately predict the MMR outcome. The training and the test scores for the linear regression were 0.54 and 0.36, respectively, which are only moderate, and not particularly for the test. To conclude, the model is not strong or weak, and for this reason, predictions of MMR with the selected features would be moderately confident.
